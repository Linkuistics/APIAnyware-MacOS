name: add-target
description: Scaffold a new target across the entire APIAnyware matrix. Usage: /add-target <name>
---

You are scaffolding a new language+paradigm target across the APIAnyware matrix.

## 1. Parse arguments

Extract `name` — the target name (kebab-case, e.g., `haskell-monadic`).

## 2. Create target directory

Create `generation/targets/{name}/` with:
- `CLAUDE.md` — using the target-level template:

```markdown
# Target: {name}

{One-line description — ask the user.}

## Before working here

1. Read `knowledge/targets/{name}.md` for target-wide learnings
2. Read `LLM_CONTEXT/coding-style.md` for coding conventions
3. Check `LLM_STATE/plans/{name}/plan.md` for current progress

## Target structure

- `crate/` — Rust emitter crate (emit-{name}) — at `generation/crates/emit-{name}/`
- `runtime/` — target-language runtime modules
- `generated/` — output (regenerated by emitter, don't edit)
- `apps/` — sample app implementations
- `tests/` — language-side smoke tests
- `test-results/` — TestAnyware evidence
```

- `apps/CLAUDE.md` — using the apps-level template (substitute {name} for target)
- `runtime/` (empty directory)
- `generated/` (empty directory)
- `tests/` (empty directory)
- `test-results/` (empty directory)

## 3. Create knowledge file

Create `knowledge/targets/{name}.md`:

```markdown
# {name} — Target Learnings

## Runtime
_(not yet populated)_

## Emitter
_(not yet populated)_

## Dated Discoveries
_(none yet)_
```

## 4. Scaffold apps across matrix

For each existing app in `knowledge/apps/` (excluding `_index.md`):
1. Create `generation/targets/{name}/apps/{app}/CLAUDE.md` using per-app template
2. Create `knowledge/matrix/{app}/{name}.md` with empty template

## 5. Create plan

Create `LLM_STATE/plans/{name}/plan.md` from `LLM_STATE/plans/plan-template.md`.

## 6. Create emitter crate stub

Create `generation/crates/emit-{name}/` with:
- `Cargo.toml` (following the emit-racket-oo pattern, with the new crate name)
- `src/lib.rs` (stub implementing LanguageEmitter with empty EmitResult)

Add to workspace members and dependencies in root `Cargo.toml`.
Add dependency and registration in `generation/crates/cli/`.

## 7. Build and verify

Run `cargo build --workspace` to verify the stub compiles.

## 8. Report

List everything created. Suggest next steps:
- Fill in the target CLAUDE.md description
- Start implementing the emitter
- Run `/begin-work {name}` to start work
